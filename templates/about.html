About Heart Disease Analysis Project

Project Overview

Heart Disease Analysis is a comprehensive data-driven platform designed to analyze and visualize heart disease data to help understand risk factors and patterns related to cardiovascular health.

Technologies Used

  • Frontend: HTML5, CSS3, JavaScript
  • Backend: Python with Flask Web Framework
  • Database: MySQL for storing patient health data
  • Data Processing: Pandas for data manipulation
  • Data Visualization: Tableau for interactive dashboards

How We Built This Project

  1. Data Collection: Collected heart disease patient data including demographics, lifestyle factors, and medical conditions
  2. Data Processing: Used Python and Pandas to clean, transform, and prepare the data for analysis
  3. Database Setup: Created MySQL database and tables to store the processed data
  4. Web Development: Built a Flask web application with responsive HTML/CSS templates
  5. Visualization: Created interactive Tableau dashboards to display key insights
  6. Deployment: Configured the application with Gunicorn for production deployment

Key Features

  • Interactive Tableau dashboards showing heart disease trends
  • Data analysis by age, gender, and lifestyle factors
  • Visual storytelling of heart disease risk factors
  • Real-time data insights for healthcare decision-making

Heart Disease Factors Analyzed

Our analysis examines multiple factors that contribute to heart disease:

  • Demographics: Age, gender, race
  • Lifestyle: Smoking, alcohol consumption, physical activity
  • Medical History: Stroke, diabetes, kidney disease
  • Health Metrics: BMI, physical health, mental health, sleep time
  • Other Conditions: Asthma, skin cancer

Goals

This project aims to:

  • Increase awareness about heart disease risk factors
  • Support early detection of cardiovascular issues
  • Provide data-driven insights for better healthcare decisions
  • Help identify patterns and trends in heart disease cases
